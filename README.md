# ZConnect Portal

This is the ZConnect Portal project.

## Backend Setup

1. Copy `.env.example` to `.env` in the `backend/` folder:
cp backend/.env.example backend/.env
---------------------------------------------------------------------------------------------
2. Running Backend

cd backend
npm install
npm start
-------------------------------------------------------------------------------------------
3. Running Frontend

cd src
npm install
npm start
---------------------------------------------------------------------------------
Notes
-Make sure your database server is running and accessible with the credentials you set in .env.
-This project uses React for frontend and Node.js for backend.
